Voice Biometric Authentication System

This web application demonstrates a voice biometric authentication system powered by an ECAPA-TDNN deep learning model. Users can register their voice samples to create a unique voice profile and later authenticate themselves by comparing a new voice sample with the stored biometric embedding using cosine similarity.

About the Project

The purpose of this application is to demonstrate how deep learning can be used for secure voice biometric authentication through a simple web interface built with Flask.
